I found a config file that logs data to the accounting system without<br />filtering it much, it basically filters by protocol, I just take tcp<br />and pass it to the flow accounting module. You could add udp and other protocols if needed.<br /><br />This is installed using<br /><br /><pre><font face="courier"><br /># ipqosconf -a exacct.qos<br /></pre></font><br />Where exacct.qos is written as follows<br /><pre><font face="courier"><br />fmt_version 1.0<br /><br />action {<br />        module ipgpc<br />        name ipgpc.classify<br />        params {<br />                global_stats true<br />        }<br />        filter {<br />                name tcpfilter<br />                protocol tcp<br />                class allclass<br />        }<br />        class {<br />                name allclass<br />                next_action acct<br />                enable_stats true<br />        }<br />}<br />action {<br />        module flowacct<br />        name acct<br />        params {<br />                global_stats true<br />                timer 10000<br />                timeout 10000<br />                max_limit 2048<br />                next_action continue<br />        }<br />}<br /></font></pre>
